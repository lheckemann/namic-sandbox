To compile the vtkPacs it is necessary to link against the dcmtk libraries.
The procedure is described next:

1.- Download the Dcmtk libraries from http://dicom.offis.de/dcmtk.php.en

2.- Untar
    $ tar -zxvf dcmtk-3.5.3.tar.gz

3.- Patch next files:

    - dcmnet/apps/findscu.cc
        <  responseIdentifiers->print(COUT);
        >  ofstream examplefile ("output_scu.txt",ios::app);
        >  responseIdentifiers->print(examplefile);
        >  examplefile.close();

        <  main(int argc, char *argv[])
        >  main_find(int argc, char *argv[])

    - dcmnet/apps/echoscu.cc
        <  main(int argc, char *argv[])
        >  main_echo(int argc, char *argv[])

    - dcmnet/apps/movescu.cc
        <  main(int argc, char *argv[])
        >  main_move(int argc, char *argv[])

    - dcmnet/apps/storescu.cc
        <  main(int argc, char *argv[])
        >  main_storescu(int argc, char *argv[])

    - dcmnet/apps/storescp.cc
        <  main(int argc, char *argv[])
        >  main_storescp(int argc, char *argv[])

4.- Configure
    $ ./configure --without-openssl --without-libtiff --without-zlib
--without-libpng --without-libxml --without-libwrap

5.- Patch the Makefile to get a library
    - dcmnet/apps/Makefile
       < echoscu: echoscu.o
       <     $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -o $@ $@.o $(LOCALLIBS)
       <     $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       < storescu: storescu.o
       <      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -o $@ $@.o $(LOCALLIBS)
       <      $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       < storescp: storescp.o
       <      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -o $@ $@.o $(LOCALLIBS)
       <      $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       < findscu: findscu.o
       <      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -o $@ $@.o $(LOCALLIBS)
       <      $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       < movescu: movescu.o
       <      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -o $@ $@.o $(LOCALLIBS)
       <      $(MATHLIBS) $(LIBS)
       > echoscu: echoscu.o
       >      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -shared -o libecho.so $@.o
       >      $(LOCALLIBS) $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       > storescu: storescu.o
       >      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -shared -o lib$@.so $@.o
       >      $(LOCALLIBS) $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       > storescp: storescp.o
       >      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -shared -o lib$@.so $@.o
       >      $(LOCALLIBS) $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       > findscu: findscu.o
       >      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -shared -o libfind.so $@.o
       >      $(LOCALLIBS) $(DCMTLSLIBS) $(OPENSSLLIBS) $(MATHLIBS) $(LIBS)
       > movescu: movescu.o
       >      $(CXX) $(CXXFLAGS) $(LDFLAGS) $(LIBDIRS) -shared -o libmove.so $@.o
       >      $(LOCALLIBS) $(MATHLIBS) $(LIBS)

6.- Make

7.- Allocate the dictionary. You may change the destination directory by setting 
the variable DCMDICPATH.
    $ cp dcmdata/libsrc/dicom.dic /usr/local/dicom/lib/dicom.dic
